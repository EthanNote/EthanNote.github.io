# E酱不喜欢键盘

### 一、 E酱想要一个好用的输入法

E酱觉得打字实在太累了，手累，键盘也一定觉得累。

于是E酱想，要是有一种能减少键盘使用量的输入法就好了，E酱打算学习编程，自己开发一个输入法。

E酱学会了哈夫曼编码算法，也编写出了基于它的输入法，理论和实践都证明了使用这个输入法打字只需要敲很少的键。

只不过，谁他喵的能记住每个字的编码呀。

E酱猛然意识到五笔字型输入法不仅编码效率高，还很好记，才明白自己在造轮子。

但是，E酱并没有泄气，打算开发一个只需要一根手指头就能熟练打字的输入法。

后来E酱发现自己不知什么时候已经精通了莫尔斯电报。



### 二、 本来只是想换个键盘

E酱打算继续学习编程，虽然不喜欢键盘打字，觉得太累，但无法抗拒编程带来的快感。

“要不买个机械键盘吧。”E酱想，笔记本电脑的键盘的手感确实不太好，买个好的键盘说不定手就不那么累了。

E酱买了个青轴机械键盘，虽然在宿舍用容易吵到舍友，但比起某舍友高亢的 DOTA 叫喊声并不算什么。况且一般也只在白天用。

手感真的很棒！

只是，键盘摆放的位置实在很尴尬，放在哪里都不如笔记本自带键盘的位置顺手。

“要不买个显示器吧。”E酱觉得显示器能让键盘用得更好，尤其是尝试过把宿舍客厅的液晶电视拿来当做显示器之后。

参考了已经买过显示器的土豪室友们之后，E酱买了个20.5寸的液晶显示器，画面比电视好很多。还好E酱平时不怎么花钱，不然就得吃土了。

用了一段时间之后，E酱发现桌子已经乱的无法收拾了，显示器、键盘、笔记本占用了大量面积，再加上数不清的数据线、电源线，还有几个U盘，移动硬盘，本来就有整洁癖的E酱不能更崩溃了。

这时，去取快递的室友回来了，带回来一台新的 Mac，据说手感、性能、操作方式都很赞，跟室友整齐的书架放在一起简直完美搭配。

“要不……”



### 三、 敲 Windows 命令真的烦

自从抛弃了整洁癖，E酱的书桌、床、衣服就全都是乱糟糟的。但电脑操作系统确越来越干净，干净到 Windows 桌面上只放了一个图标。

“因为这样可以加快开机速度。”E酱向室友解释。实际上是因为桌子上没有足够的空间用来给鼠标滑动了。

E酱不得不想办法尽量用键盘取代鼠标，网上搜了一大堆 CMD 命令的使用方法，遇到通常需要鼠标点击来完成的操作，就直接敲几个命令来完成，熟练之后反而比鼠标还快。

后来E酱用上了 Linux 系统，bash 解决一切问题。E酱觉得 vim 真的是个良心发明，完全不需要鼠标，熟练之后打字飞快。真想不明白微软是如何提高用户体验的。

除了打游戏的时候。
